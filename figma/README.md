EventEase Figma Import Pack

Contents
- tokens.json: Tokens Studio-compatible design tokens
- sitemap.svg: App sitemap diagram
- flow-create-event.svg: Create Event flow
- wireframe-dashboard.svg: Low-fi dashboard wireframe
- wireframe-events.svg: Low-fi events list wireframe
- wireframe-event-creation.svg: Low-fi event creation flow wireframe

How to import
1) Create a new Figma file. Add pages: Sitemap, Flows, Wireframes, Components.
2) Drag each .svg from this folder into the canvas on the matching page.
3) For tokens.json:
   - Install the "Tokens Studio for Figma" plugin.
   - Open the plugin → Import → Choose tokens.json.
   - Apply tokens to Text Styles and Color Styles via the plugin.

Recommended setup
- Desktop frame: 1440×1024, 12-col grid, 80px margins, 24px gutters.
- Type: Inter (H1 32, H2 24, H3 18, Body 16, Small 14).
- Spacing scale: 4, 8, 12, 16, 24, 32, 48, 64.

Notes
- SVGs are intentionally low-fidelity to speed iteration.
- Convert repeated sections into Components after placing (Navbar, Sidebar, Cards, Tables, Buttons).
